## Take a Look
Link 👉 https://doubtout.netlify.app/

# Features
### User side
- ask the question
- give the answer
- upvote and downvote
- their profile Analysis

### Admin Side
- see all user & delete any.
- see all question & delete any.
- see all answer & delete any.
- whole system analysis

## How To Run ?
**for frontend** <br>
```sh
  npm i
  npm start
```


**for backend** <br>
 
```sh
 cd backend 
 npm i
 nodemon ./index.js
```
